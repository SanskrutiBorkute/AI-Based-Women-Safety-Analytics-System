## Folder Description

### data/

Contains all datasets and the SQLite database used by the application.

* **crime_data.csv** вЂ“ Crime dataset used for AI safety prediction.
* **safety_system.db** вЂ“ SQLite database storing user accounts, trusted contacts, user settings, activity history and authentication data.

---

### models/

Contains the trained Machine Learning model.

* **safety_model.pkl** вЂ“ Random Forest model used to predict the safety level (Safe, Medium Risk or High Risk).

---

### static/

Stores all frontend resources.

#### css/

* **style.css** вЂ“ Complete styling of the application including dashboard, authentication pages, cards, buttons, map, analytics and responsive layouts.

#### images/

Stores project images, icons and other visual assets.

#### js/

* **main.js** вЂ“ Main JavaScript file containing:

  * Live location tracking
  * AI safety prediction
  * AI risk analysis
  * Trusted contacts
  * Nearby support network
  * Emergency SOS
  * Voice SOS
  * Safe route recommendation
  * Notifications
  * Analytics
  * Dashboard interactions
  * Authentication support

---

### templates/

Contains all HTML templates rendered by Flask.

* **auth.html** вЂ“ Login and Registration page.
* **base.html** вЂ“ Base template shared across pages.
* **dashboard.html** вЂ“ Main dashboard displaying all AI-powered safety features.

## Python Files

### app.py

Main Flask backend application handling:

* User authentication
* Dashboard rendering
* AI prediction
* Database operations
* SOS workflow
* APIs
* Session management

### generate_large_dataset.py

Generates the expanded crime dataset used for AI model training.

### init_db.py

Creates and initializes the SQLite database.

### train_model.py

Trains the Machine Learning model and generates **safety_model.pkl**.
