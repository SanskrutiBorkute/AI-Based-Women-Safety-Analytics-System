## Project Description

SafeHer is an AI-powered Women Safety Analytics System developed using Flask, Python, Machine Learning, SQLite, HTML, CSS, and JavaScript. The application helps women analyze the safety level of their current location by using live GPS location, historical crime data, machine learning predictions, nearby emergency services, and intelligent safety recommendations.

The system also provides emergency features such as SOS alerts, Voice SOS, trusted contacts, nearby police stations, hospitals, AI chat assistant, analytics dashboard, and safe route recommendations.

Ask safety-related questions such as:

# Deployment on Render

1. Push the complete project to GitHub.

2. Login to Render.

3. Create a new **Web Service**.

4. Connect your GitHub repository.

5. Build Command:

```bash
pip install -r requirements.txt
```

6. Start Command:

```bash
gunicorn app:app
```

7. Click **Deploy**.

8. After deployment, Render will provide a live URL for accessing the application.

# Troubleshooting

### If location is not detected

* Allow browser location permission.
* Refresh the page.

### If the map is not loading

* Check your internet connection.
* Refresh the browser.

### If dependencies are missing

Run:

```bash
pip install -r requirements.txt
```

### If the server does not start

Verify that Python is installed correctly and run:

```bash
python app.py
```
